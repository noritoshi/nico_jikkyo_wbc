# AI コメント生成機能

## 概要

Gemini APIを使ってコメントを生成し、ニコニコ生放送に投稿する機能。

## 認証方式

- ユーザーがGoogle AI StudioでAPI Keyを取得（無料枠: 1分15リクエスト）
- ポップアップの設定欄にAPI Key入力 → chrome.storage.localに保存
- API Key取得先: https://aistudio.google.com/api-keys

## 対応する方式（プリセット）

### 1. 通常コメント
- 1行の流れるコメント
- 単発postComment送信

### 2. 下積みCA
- shitaで下から積み上げるアスキーアート
- 各行をshitaコメントとして100ms間隔で連続送信
- 最大8行程度

## UI設計

### ポップアップ
- 既存UIの下に「設定」セクション追加
- Gemini API Key入力欄（chrome.storage.localに保存）

## Geminiプロンプト

### 通常コメント用

```
あなたはニコニコ生放送の実況コメントを生成するアシスタントです。

現在の番組の直近のコメント:
{recentComments}

ユーザーの依頼: {userPrompt}

以下のルールに従ってコメントを1つ生成してください:
- 最大75文字
- ニコニコ生放送の実況らしい自然な口調
- 絵文字は使わない
- コメント本文のみを返してください（説明不要）
```

### 下積みCA用

```
あなたはニコニコ生放送のコメントアート職人です。
「下積み」方式でコメントアートを作成してください。

下積みの仕組み:
- 各行が「shita」コマンドで下から上に積み上がる
- 1行目が最下段、最終行が最上段に表示される
- 各行は最大75文字（全角）
- 等幅ではない（MSPゴシック）ので、全角文字・記号で幅を揃える
- 空白にはU+3000（全角スペース）を使う
- 最大8行程度が実用的

ユーザーの依頼: {userPrompt}

以下の形式で出力してください（各行がshitaコメントとして投稿されます）:
1行目（最下段）
2行目
...
最終行（最上段）

コメントアートのみを出力し、説明は不要です。
```

## 実装アーキテクチャ

```
content_script.js (AIパネルUI + プレビュー)
    ↓ chrome.runtime.sendMessage
background.js (API Key取得 + Gemini API呼び出し)
    ↓ レスポンス
content_script.js (プレビュー表示)
    ↓ 投稿ボタン
offscreen.js (WebSocket経由でpostComment送信)
```

- Gemini API呼び出しはbackground.js（Service Worker）で行う
  - content_scriptからはCORS制限でAPI呼び出し不可
  - offscreen.jsでも可能だが、background.jsの方がシンプル
- API Key はchrome.storage.localから取得
- 直近のコメントはcontent_script側で保持（最新20件程度）

## 投稿処理

| 方式 | 処理 |
|------|------|
| 通常コメント | 1回のpostComment送信 |
| 下積みCA | 各行をshita+色指定で100ms間隔連続送信 |
